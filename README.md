# Versioning Gradle Plugin

## Setup

Using the [plugins DSL](https://docs.gradle.org/current/userguide/plugins.html#sec:plugins_block):

``` kotlin
plugins {
    id("io.github.anvell.versioning.gradle.plugin") version "0.1.0"
}
```
